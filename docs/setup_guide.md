# Development Setup Guide

## Prerequisites
- Python 3.8+
- Git (for version control)

## First-Time Setup
```bash
git clone https://github.com/your-repo/expense-tracker.git
cd expense-tracker
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
# OR
.venv\Scripts\activate     # Windows

pip install -r requirements.txt
